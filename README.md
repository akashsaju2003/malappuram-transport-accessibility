&#x20;Public Transport Accessibility Study – Malappuram



&#x20;Overview

This project measures how accessible bus stops are to residents of Malappuram district.  

Using Python (GeoPandas, Rasterio) and QGIS, we mapped bus stop buffers and compared them with population data to identify served and underserved areas.



\---



&#x20;Key Results

\- Total Population: 3,178,661  

\- Served Population: 518,565 (27.77%)  

\- Unserved Population: 2,660,096 (72.23%)  



 Only about one‑quarter of Malappuram’s population lives within 500 m of a bus stop.



\---



&#x20;Outputs

\- Jupyter Notebook with full workflow  

\- Shapefile of bus stop buffers  

\- Raster of unserved population  

\- Coverage summary text file  

\- Maps showing served vs. unserved areas



\---



The study highlights major accessibility gaps in public transport.  

The unserved population maps point to priority zones where new bus stops could improve equity and mobility.

\## How to Reproduce

1\. Open the notebook in Jupyter.  

2\. Install required libraries (`geopandas`, `rasterio`, `rasterstats`, `osmnx`).  

3\. Run cells step by step.  

4\. Load outputs into QGIS for visualization.
